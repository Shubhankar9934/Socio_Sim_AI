JADU (v0.1.0) – Synthetic society simulation: survey orchestration, analytics, evaluation.
Flow: 1) POST /population/generate → 2) GET /agents (optional) → 3) POST /survey or POST /survey/multi → 4) GET /analytics/:survey_id, POST /evaluate/:survey_id. Simulation, discovery, calibration can be used as needed.
Env: base_url, session_id, survey_id, agent_id. See postman/README.md for full input/output and flow per API.
